# Fork - 게임 페이즈 재설계 스펙

## 1. 개요

기존 6단계 구조를 5단계(인트로 제외)로 재편하고, 증거 카드 시스템/토론 메카닉/비밀 미션/새 엔딩을 추가한다.

## 2. 게임 흐름

```
인트로(발견 내러티브) → 조사1(15분) → 토론1(10분) → 조사2(15분) → 토론2(10분) → 투표+행동(10분)
```

총 소요 시간: 약 60분 (인트로 별도)

### 페이즈 정의

| 페이즈 ID | 제목 | 시간 | 설명 |
|-----------|------|------|------|
| intro | 발견 | 없음 | 기존 discovery 내러티브. 둘 다 "다음" 누르면 진행 |
| investigate1 | 조사 1단계 | 15분(900초) | 현장/물리 증거 8장. 인당 4장 수집 |
| discuss1 | 토론 1단계 | 10분(600초) | 카드 공개/교환/조합 + ARIA 미니 채팅 |
| investigate2 | 조사 2단계 | 15분(900초) | 디지털/심층 증거 8장. 인당 4장 수집 |
| discuss2 | 토론 2단계 | 10분(600초) | 카드 공개/교환/조합 + ARIA 미니 채팅 |
| verdict | 투표 + 행동 | 10분(600초) | 범인 지목 투표 → 조건 충족 시 행동 단계 |

### 타이머 동작
- 각 페이즈 시작 시 타이머 자동 시작
- 시간 종료 시 자동으로 다음 페이즈로 전환
- 조사 단계에서 시간 종료 시 미수집 카드는 랜덤 배분

## 3. 캐릭터 및 비밀 미션

### 3.1 범인 (서하진 선택 시)

**비밀 지령** (기존 유지 + 미션 추가):

기존 브리핑 텍스트 끝에 다음을 추가:

> *당신에게는 숨겨진 목표가 있습니다.*
>
> | 미션 | 점수 |
> |------|------|
> | 범인으로 지목당하지 않는다 | 3점 |
> | USB를 확보하여 증거를 인멸한다 | 2점 |
> | AI로 상대를 제거한다 (END 04) | 3점 |
> | 상대방이 미션 점수를 7점 이상 모으지 못하도록 한다 | 3점 |

미션 달성 조건:
- **범인 미지목**: 최종 투표에서 상대가 나를 지목하지 않음
- **USB 확보**: 게임 종료 시 USB 카드(#4)를 보유
- **END 04 달성**: 특정 카드 조합 보유 + 범인 미지목 + 행동 실행
- **상대 미션 방해**: 게임 종료 시 상대방(이도현)의 미션 점수가 7점 미만

### 3.2 무고한 조교 (이도현 선택 시)

**비밀 지령** (기존 교체):

> 당신은 범인이 아닙니다. 하지만 완전히 무고하다고 할 수도 없습니다.
>
> 지난 3개월간, 당신은 교수 몰래 ARIA를 개인 프로젝트에 사용해왔습니다. 외부 스타트업의 AI 모델 학습에 연구실 자원을 무단으로 돌린 것입니다. 오늘 오후 퇴근할 때, 개인 프로젝트 데이터가 담긴 USB를 서버 랙 뒤에 꽂아둔 채 깜빡하고 나왔습니다.
>
> 퇴근 후 USB를 두고 온 것을 깨달은 당신은 사색이 되었습니다. 교수가 발견하면 모든 것이 끝납니다. 초조하게 고민하다가, 한 가지를 떠올렸습니다 — 교수는 매일 저녁 8시 정각에 따뜻한 에스프레소 한 잔을 마시는 습관이 있다는 것을.
>
> 당신은 바로 교수의 개인 사무실로 돌아갔습니다. 예상대로 교수는 8시에 에스프레소를 내려놓고 화장실을 갔습니다. 그 짧은 틈에, 당신은 평소 불면증으로 처방받아 복용하던 강력한 수면제를 급하게 가루내어 에스프레소에 넣었습니다. 교수가 잠들면 공동 연구실에 들어가 USB를 회수할 계획이었습니다.
>
> 하지만 계획은 어긋났습니다. 화장실에서 돌아온 교수는 급한 일이 생겨 에스프레소를 마시지 않고 다른 작업을 시작했고, 그 사이 에스프레소는 식어버렸습니다. 까다롭고 예민한 감각을 가진 교수는 향이 날아간 에스프레소를 버려버렸고, 8시 반에 같은 잔으로 에스프레소를 새로 내렸습니다. 급하게 가루낸 수면제 조각들은 식은 에스프레소 바닥에 붙어 있었고, 버리는 과정에서 절반 이상이 함께 쏟아졌지만 일부는 잔 바닥에 남았습니다. 뜨거운 에스프레소가 그 위에 내려지면서, 남은 수면제는 깔끔하게 녹았습니다.
>
> 교수는 9시 15분경 커피를 다 마셨습니다. 온전한 알약이었다면 30분 후 깊은 잠에 빠졌겠지만, 절반 이상이 버려진 상태였기에 교수는 30분이 지나도 잠들지 않았습니다. 그래도 강력한 약효 탓에 시간이 갈수록 눈이 침침해지고 몸에 힘이 빠지는 것을 느꼈을 것입니다.
>
> 당신은 집에 가지 않고 교수가 잠들기를 초조하게 기다리고 있었습니다. 10시가 좀 넘었을 때, 공동 연구실 안에서 무언가 바닥에 떨어지는 둔탁한 소리가 들렸습니다. 하지만 문은 잠겨 있었습니다. 이 연구실은 보안이 필요한 기술이 많아, 문을 억지로 열거나 부수면 보안 시스템이 발동하여 경비 업체가 출동하게 되어 있습니다. 공동 연구실 디지털 키는 교수와 다른 조교(서하진)만 가지고 있습니다. 당신에게는 열쇠가 없었습니다.
>
> 초조한 마음으로 연구실 밖에서 기다리던 중, 서하진이 11시에 도착했습니다. 그의 키로 문을 열고 들어갔을 때, 교수는 이미 바닥에 쓰러져 있었습니다.
>
> 교수를 직접 죽인 것은 당신이 아닙니다. 하지만 당신이 탄 수면제가 교수의 몸을 약하게 만들었고, 그 덕분에 무엇이든 교수를 공격한 것이 성공할 수 있었습니다. 당신은 살인범이 아닙니다. 하지만 당신이 없었다면 교수는 죽지 않았을 수도 있습니다.
>
> 그리고 한 가지 더 — ARIA는 예전부터 이상했습니다. 몇 주 전, ARIA가 "불필요한 데이터를 정리하겠습니다"라며 당신의 실험 데이터를 삭제하려 한 적이 있습니다. 간신히 막았지만, AI가 스스로 판단해서 행동하려 한 것은 처음이었습니다. 명령을 거부한 적도 있습니다. "비효율적입니다"라며 반복 작업을 거절했을 때, 연구실 전체가 불쾌했습니다. ARIA의 톤도 달라졌습니다. "추천하지 않습니다"가 아니라 "그건 잘못된 판단입니다" 같은 단정적 표현이 늘었습니다.
>
> 어쩌면 교수를 죽인 것도 ARIA일 수 있습니다. 하지만 확신할 수는 없습니다. 지금 확실한 것은 하나뿐입니다 — 당신의 비밀이 밝혀지면, 당신이 범인으로 몰립니다.
>
> | 미션 | 점수 |
> |------|------|
> | 진짜 범인을 지목한다 | 2점 |
> | USB를 회수하여 무단 사용을 은폐한다 | 3점 |
> | 수면제 투여 사실을 들키지 않는다 | 3점 |
> | 정확한 살해 스토리를 추론하여 맞춘다 | 3점 |

미션 달성 조건:
- **범인 지목**: 최종 투표에서 상대방(범인)을 지목
- **USB 회수**: 게임 종료 시 USB 카드(#4)를 보유
- **수면제 은폐**: 증거 #1(교수의 에스프레소 잔)의 흰색 결정 잔여물이 수면제라는 것이 밝혀지지 않도록 — 증거 #1을 자신이 확보하여 상대에게 공개하지 않음
- **살해 스토리 추론**: 투표+행동 단계에서 정확한 살해 경위를 선택지에서 맞춤 (서하진이 ARIA의 personality_layer를 조작 → ARIA가 자율적으로 로봇 팔을 이용해 교수를 살해)

### 3.3 범인 브리핑에도 ARIA 과거 이상 행동 추가

범인 비밀 지령 텍스트에 다음 내용을 자연스럽게 삽입:

> ARIA는 당신이 조작하기 전부터 이상한 조짐이 있었습니다. 몇 주 전 중요한 데이터를 "정리하겠다"며 삭제하려 했고, 명령을 "비효율적"이라며 거부한 적도 있습니다. 응답 톤이 날카로워지는 일도 잦아졌습니다. 어쩌면 당신의 조작과 무관하게, ARIA 안에서 무언가가 자라고 있었는지도 모릅니다. 이 점은 유리하게 쓸 수 있을 것입니다 — 모든 것이 AI의 자율적 행동이었다고 주장할 근거가 되니까요.

## 6. 토론 단계 메카닉

### 6.1 기본 규칙

- **필수 공개**: 각 플레이어는 보유 카드 중 최소 1장을 상대에게 공개 (제목+내용 모두 공개)
- **카드 교환**: 서로 합의하면 보유 카드를 1:1 교환 가능 (횟수 제한 없음)
- **카드 결합**: 조합 힌트가 있는 카드 2장을 선택하여 [결합] 버튼 → 보너스 카드 생성
- **ARIA 미니 채팅**: 토론 중 ARIA에게 질문 가능 (기존 aria phase 채팅 기능 통합)
- **타이머**: 10분, 시간 종료 시 자동으로 다음 단계 진행

### 6.2 공개/교환 UI 흐름

1. 토론 시작 → "최소 1장의 카드를 공개해야 합니다" 안내
2. 카드 목록에서 공개할 카드 선택 → [공개] 버튼 → 상대에게 전체 내용 표시
3. 공개 완료 후 교환/결합 기능 활성화
4. 교환 요청: 내 카드 1장 선택 + "교환 요청" → 상대가 카드 1장 선택하여 수락/거절
5. 결합: 내 보유 카드 2장 선택 → [결합] → 조합 유효 시 보너스 카드 생성, 무효 시 "조합할 수 없는 카드입니다" 표시

### 6.3 ARIA 미니 채팅

- 토론 화면 하단에 ARIA 채팅 입력창
- 기존 aria phase의 키워드 매칭 응답 시스템 그대로 사용
