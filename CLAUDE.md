# Project Rules

## ABSOLUTE DO NOT TOUCH

game-data.js 내의 아래 항목들은 커맨더가 직접 명령하기 전까지 절대로 임의로 건드리지 않는다:

- **prologue > narrative > shared** (사건 개요 공통 내러티브)
- **roles > culprit > briefing** (하진 비밀지령)
- **roles > innocent > briefing** (도현 비밀지령)

이 세 항목은 어떤 이유로든, 리팩터링/개선/버그수정/줄바꿈 조정/시간 표기 변경/문체 수정 명목이든, 커맨더의 직접적이고 명시적인 요청 없이는 한 글자도 변경 금지.

**다른 부분을 수정하는 과정에서도 이 세 항목은 절대 건드리지 않는다.** 주변 코드 구조 변경, 포맷팅, 마이그레이션, 어떤 작업이든 이 세 항목의 내용은 현재 상태 그대로 보존해야 한다. 커맨더가 직접 "이 항목을 바꿔라"라고 말하기 전까지는 읽기 전용으로 취급한다.

## 사건개요/비밀지령 현재 확정 내용 (2026-05-10 확정)

아래는 현재 확정된 내용의 요약이다. 원본은 game-data.js에 있으며, 절대 임의 수정 금지.

### 사건개요 (prologue > narrative > shared)
- "(하진이 읽어주세요)"로 시작, "(도현이 읽어주세요)"로 전환
- 대사는 별도 문단이 아닌 본문과 같은 줄에 인라인 배치
- "하지만 역시 오래 갈 리가 없었다." (그런데 아님)
- 교수 대사: "다들 오늘은 이쯤 하고 들어가서 쉬어라."
- prologue에는 shared만 존재. role-specific(culprit/innocent) 내러티브 없음
- server.js에서 prologue 전송 시 shared만 전송 (buildNarrative 사용 금지)

### 하진 비밀지령 (roles > culprit > briefing)
- "당신은 범인입니다."로 시작
- ---RED--- / ---WHITE--- 구분자로 비밀정보와 미션 목표 구분
- 미션: 범인 지목 회피(3점), USB 보유(3점), AI로 상대 제거(3점), 상대 점수 방해(3점)

### 도현 비밀지령 (roles > innocent > briefing)
- "당신은 직접적인 범인은 아닙니다."로 시작
- ---RED--- / ---WHITE--- 구분자로 비밀정보와 미션 목표 구분
- "당신의 비밀은 숨기고, 진짜 사건의 전말을 알아내야 합니다."
- 미션: 진범 지목(3점), USB 보유(3점), 수면제 은폐(3점), 살해 스토리 추론(3점)

## 단계별 시간 설정 (2026-05-10 확정)

| 단계 | 시간 | 비고 |
|------|------|------|
| 프롤로그 | 제한 없음 | 타이머 없음 (duration: 0) |
| 조사단계 1 | 15분 (900초) | 빨리 넘어가기 버튼 있음 (둘 다 동의 시) |
| 토론단계 1 | 10분 (600초) | 빨리 넘어가기 버튼 없음 |
| 조사단계 2 | 15분 (900초) | 빨리 넘어가기 버튼 있음 (둘 다 동의 시) |
| 토론단계 2 | 10분 (600초) | 빨리 넘어가기 버튼 없음 |
| 최종 판결 | 10분 (600초) | 빨리 넘어가기 버튼 없음 |

- 타이머 표시 형식: `남은시간 / 총시간` (예: `12:30 / 15:00`)
- 빨리 넘어가기(준비 완료) 버튼은 조사단계에만 존재
- 최종 판결 화면에도 타이머 표시
- 사건개요와 비밀지령 사이에 빨간색 안내 문구 표시 (개인 숙지 안내, 권장 15분, 협의 조정 가능)

## 타이틀 화면 디자인 (2026-05-12 확정)

- 배경: 크라프트지 텍스처 (`kraft-texture.png`)
- 제목: 캘리그래피 로고 이미지 (`Title.png`) — Fork + 갈라진 의도
- 일러스트: 코드 실루엣 인물 (`title-illustration.png`) — 화면 중앙 배치
- 일러스트 뒤 흰색 원형 후광 + 글로우 효과 (4초 주기 깜빡임 애니메이션)
- 이중 장식 테두리: 바깥 2.5px, 안쪽 1.7px (opacity 0.8), 액자/동화책 느낌
- 레이아웃 3단 구조 (space-between):
  - 상단: 캘리그래피 로고 이미지
  - 중앙: 일러스트 이미지 + 후광
  - 하단 클러스터: 2인용 머더 미스터리 + 시작하기 버튼
- 글자 색상: 크라프트지 배경에 맞춘 어두운 갈색 계열
- 시작하기 버튼: 다크 브라운 (`#3d342a`)

## 폰트 설정 (2026-05-12 확정)

- 강조/디스플레이 폰트 (`--font-display`): 부크크 명조 Bold (`BookkMyungjo`, weight 700)
- 본문 폰트 (`--font-body`): Noto Sans KR
- 모노 폰트 (`--font-mono`): IBM Plex Mono
- 타이틀 하단 클러스터 (2인용 머더 미스터리, 시작하기): 부크크 명조
- 비밀지령 안내문구: font-weight 400 (일반 굵기), ※ 기호로 시작

## 개발 편의 기능

### Dev 모드
- `?dev=screen-waiting` 등 URL 파라미터로 특정 화면을 더미 데이터와 함께 바로 열 수 있음
- 사용 가능한 화면: screen-title, screen-lobby, screen-waiting, screen-character-select, screen-intro, screen-investigation, screen-verdict, screen-ending

### FOUT 방지
- index.html에서 폰트 로드 전까지 `visibility: hidden` 처리
- `document.fonts.ready` 또는 최대 2초 후 `fonts-loaded` 클래스 추가하여 표시
